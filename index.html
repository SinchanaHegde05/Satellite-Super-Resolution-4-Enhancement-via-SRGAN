RRDB
Residual-in-Residual Dense Block
3 nested ResidualDenseBlocks per RRDB unit. Dense skip connections allow gradient flow through 16+ blocks without vanishing.
VGG19
Perceptual Loss
Feature-level loss from relu3_4 of frozen VGG19. Produces sharp textures vs plain MSE which causes over-smoothing.
GAN
Adversarial Training
PatchGAN discriminator enforces local texture realism. Loss = 1.0×perceptual + 1e-3×adversarial + 1e-2×pixel L1.
4×
Sub-pixel Upsampling
Two PixelShuffle blocks (each 2×) avoid checkerboard artifacts common in naive transposed convolution upsampling.
